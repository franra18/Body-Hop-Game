# Body Hop

Bienvenido al repositorio de **Body Hop**, un proyecto de videojuego desarrollado en Unity.

## 🚀 Exploración en Escena Inicial

Para facilitar las pruebas de diseño y nivel, se ha implementado una funcionalidad de **Cámara Libre** en la escena inicial.

- **Al ejecutar el juego:** Podrás moverte libremente por el escenario sin las restricciones del controlador del jugador.
- **Utilidad:** Esto permite inspeccionar la profundidad del río, el ajuste del puente sobre el terreno y las colisiones desde cualquier ángulo.

## 🛠️ Detalles del Proyecto

Actualmente, el proyecto cuenta con:
* **Sistema de Terreno:** Configurado con `Terrain Collider` optimizado.
* **Diseño de Niveles:** Puentes que conectan plataformas con un río de profundidad variable en la zona inferior.
* **Optimización de Assets:** Gestión selectiva de paquetes para mantener el proyecto ligero.

## 🎮 Controles de Cámara (Play Mode)
* **Mouse:** Control de rotación (Mirar alrededor).
* **WASD:** Movimiento direccional.
* **Q / E:** Subir o bajar la altura de la cámara.
